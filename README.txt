Anleitung zum Ausführen des Computer-Info-Tools:

1. Kopieren Sie diese Dateien in ein Verzeichnis:
   - main.ps1 (Das Computer-Info-Tool)
   - DTCertificate.cer
   - Install-Certificate.ps1

2. Führen Sie PowerShell als Administrator aus

3. Navigieren Sie zum Verzeichnis mit den kopierten Dateien

4. Führen Sie zuerst das Installer-Skript aus:
   .\Install-Certificate.ps1

5. Nach erfolgreicher Installation können Sie das Computer-Info-Tool ausführen:
   .\main.ps1
